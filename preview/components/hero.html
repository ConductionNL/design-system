React / MDX usage

Mount in MDX as <Hero/>

The React component lives in @conduction/docusaurus-preset/components and mirrors this kit page exactly. Use it on any Docusaurus site that imports the preset.

Import & example

import {Hero} from '@conduction/docusaurus-preset/components';

<Hero
  eyebrow={<>Twelve open-source apps · one <span className="next-blue">Nextcloud</span></>}
  title={<>Install your stack.<br/>In two minutes.</>}
  lede="Twelve open-source apps that plug into Nextcloud..."
  primaryCta={{label: 'Install from app store', href: '/install'}}
  secondaryCta={{label: 'Get a demo via a partner', href: '/partners'}}
  tertiaryCta={{label: 'View on GitHub', href: 'https://github.com/...'}}
>
  {/* Right-column visual: hex-rain mini-game, platform diagram, anything that fills the column */}
  <div className="hex-rain" />
</Hero>

Props

  • eyebrow Plex Mono caption above the title
  • title H1, string or ReactNode (supports br)
  • lede body paragraph below the title
  • primaryCta {label, href} cobalt fill
  • secondaryCta {label, href} bordered ghost
  • tertiaryCta {label, href} arrow link
  • meta {primary, secondary} micro-caption
  • children right-column visual slot
