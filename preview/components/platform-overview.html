The platform overview

Five components. Twenty-four apps. One Nextcloud workspace.

Every Conduction app slots into one of five component-cards. All of them run on the Nextcloud platform — files, mail, calendar, identity — that your team already trusts. Hover any app to see what it does.

OpenRegisterData
Schema-driven object store with audit trail. The data foundation underneath every workplace app.
OpenConnectorIntegrate
API gateway for everything outside Nextcloud — REST, SOAP, GraphQL, webhooks. Maps and transforms between systems.
DocuDeskDocs
Generate, anonymise, sign, and template documents. The documents-and-forms layer.
NLDesign ThemeTheme
NL Design System tokens applied across every Conduction app. Government-compliant theming, drop-in.
MyDashDash
Dashboard builder with role-based access. Workflow automation across registers.
Automation
Trigger flows from any data event. Schedule jobs, fire webhooks, run scripts.
Agents
Conversational agents grounded in your registers. No vendor lock-in on the LLM.
Intelligence
Cross-register insights and anomaly detection. Privacy-first, on-prem option.
Workspace
Files
Sync, share, and version any file. The cornerstone of every Nextcloud workspace.
Mail
IMAP and SMTP mailbox right inside Nextcloud — no separate desktop client.
Calendar
CalDAV-backed shared calendars with free-busy across teams.
Talk
End-to-end-encrypted chat, voice, video — entirely on your own servers.
Office
Collabora-powered docs, sheets, and slides. Real-time co-editing, no Google.
