This is the first post on the new Conduction site. We are in active development; the brand book at [connext.conduction.nl/identity](https://connext.conduction.nl/identity/) will be re-homed here once the OpenCatalogi content plugin lands.

## What is on the site

- **Platform proposition:** what Conduction is, who it is for, how the apps fit on Nextcloud
- **Per-app docs:** separate sub-domains under `*.conduction.nl`, all built on the same `@conduction/docusaurus-preset`
- **Diagrams set:** the seven `<cn-*>` web components rendered live, with API tables and walk-throughs

## What is next

- Wire `@conduction/docusaurus-plugin-opencatalogi` so editors update content from the Nextcloud-side OpenCatalogi UI
- Translate the homepage and docs into EN, DE, FR (NL is canonical)
- Stand up `apps.conduction.nl` and `commonground.nu` on the same preset
