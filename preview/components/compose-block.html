Compose block · branded code panel for verbatim copy-paste content

Cobalt-900 panel, Plex Mono, copy-on-hover.

For docker-compose YAML, bash recipes, JSON examples, anything the reader might want to copy. Optional filename pill at the top, a copy-to-clipboard button on hover.

docker-compose.yml 
services:
  nextcloud:
    image: nextcloud:latest
    restart: unless-stopped
    ports:
      - "8080:80"
    environment:
      # Postgres connection details
      POSTGRES_HOST: db
      POSTGRES_USER: nextcloud
bash 
# one-time setup
docker compose up -d

# tail the logs
docker compose logs -f nextcloud
React / MDX usage

Mount in MDX as <ComposeBlock/>

For verbatim copy-paste blocks (docker-compose, bash, env files) on connext marketing pages where the brand-styled cobalt-900 panel reads better than Docusaurus's default Prism-themed code block. For syntax-highlighted code on doc pages, keep using the standard fenced code block.

Import & example

import {ComposeBlock} from '@conduction/docusaurus-preset/components';

<ComposeBlock filename="docker-compose.yml" lang="yaml">
{`services:
  nextcloud:
    image: nextcloud:latest
    ports: ["8080:80"]
`}
</ComposeBlock>

<ComposeBlock filename="bash">
{`docker compose up -d
docker compose logs -f nextcloud`}
</ComposeBlock>

Props

  • filename optional pill at the top (Plex Mono caps)
  • lang sets language-X class on the inner code element (no syntax-highlighting; for screen readers and consumer styling)
  • children raw text; preserve newlines with template literals

Behaviour

  • Copy button fades in on hover; click copies the inner text via navigator.clipboard
  • Copied state shows '✓ copied' for 1.6 seconds
  • Falls back to no-op if clipboard API is unavailable (iframes, no permission)
