React / MDX usage

Mount in MDX as <DetailHero/>

Two-column 1fr/360px hero shared by every detail page in the kit. Left column: crumb + status badge + version + h1 + tagline + 3 CTAs. Right column: 360x416 cobalt hex with the page-subject's icon.

Import & example

import {DetailHero} from '@conduction/docusaurus-preset/components';

<DetailHero
  crumb={[{label: 'Apps', href: '/apps'}, 'OpenRegister']}
  status={{label: 'Stable', color: 'var(--c-mint-500)'}}
  version="v3.1"
  locales="NL · EN"
  title="OpenRegister"
  tagline="Schemas, registers, structured data objects..."
  primaryCta={{label: 'Install from app store', href: '/install'}}
  secondaryCta={{label: 'Read the docs', href: '/docs/openregister'}}
  tertiaryCta={{label: 'View on GitHub', href: 'https://github.com/...'}}
  iconColor="var(--c-blue-cobalt)"
  icon={<svg viewBox="0 0 24 24">...</svg>}
/>

Props

  • crumb array of {label, href} or strings
  • status {label, color} for the leading hex-bullet badge
  • version Plex Mono caption next to the status
  • locales e.g. "NL · EN"
  • title h1, string or ReactNode
  • tagline body paragraph
  • primaryCta / secondaryCta / tertiaryCta three action buttons
  • icon ReactNode SVG, 24x24 viewBox
  • iconColor override the cobalt hex fill (e.g. white for partner logos)
