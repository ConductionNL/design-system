React / MDX usage

Mount in MDX as <SolutionCard/>

Pairs with <SolutionGrid/>. Sector tints the icon hex (public, mkb, health). Built-on pills come from a string array.

Import & example

import {SolutionGrid, SolutionCard} from '@conduction/docusaurus-preset/components';

<SolutionGrid columns={2}>
  <SolutionCard
    href="/solutions/woo"
    sector="public"
    sectorLabel="Publieke sector"
    icon={<svg viewBox="0 0 24 24">...</svg>}
    title="WOO compliance, by Friday."
    outcome="Live WOO portal at your Nextcloud..."
    builtOn={['OpenCatalogi', 'OpenRegister', 'OpenConnector']}
  />
</SolutionGrid>

Props

  • sector public | mkb | health (tint)
  • sectorLabel override the auto label
  • icon ReactNode SVG, 24x24 viewBox
  • title h3, string or ReactNode
  • outcome body paragraph
  • builtOn string array of app names
  • href link target (renders as <a>)
