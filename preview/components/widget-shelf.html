React / MDX usage

Mount in MDX as <WidgetShelf/>

Section that lists the dashboard widgets a Conduction app ships for Nextcloud. Each app that registers a Nextcloud-dashboard widget shows up automatically in MyDash; this section makes that promise concrete by drawing the widgets the app actually contributes.

Import & example

import {WidgetShelf} from '@conduction/docusaurus-preset/components';

<WidgetShelf
  eyebrow="Widgets we ship"
  title="On every Nextcloud dashboard."
  lede="Install OpenRegister and the home screen surfaces what changed..."
  widgets={[
    {
      title: 'Recent records',
      desc: 'Latest objects across every register...',
      panel: <div className="w w-records">...</div>,
    },
    /* ... */
  ]}
/>

Props

  • eyebrow Plex Mono caption above the title
  • title headline (string or ReactNode)
  • lede body copy under the title
  • widgets array of {title, desc, panel}
  • columns override (default 3 when widgets ≥ 4)
