React / MDX usage

Mount in MDX as <StatsStrip/>

Cobalt-50 band with up to four big-number stats. Auto-collapses to 2-up under 900px and 1-up under 500px.

Import & example

import {StatsStrip} from '@conduction/docusaurus-preset/components';

<StatsStrip stats={[
  {value: '12',    label: 'apps in the ecosystem, all open-source'},
  {value: '2 min', label: 'from app store install to working solution'},
  {value: '€0',    label: 'for the apps, support is optional'},
  {value: 'EUPL',  label: 'licensed, no vendor lock-in, ever'},
]} />

Props

  • stats array of {value, label}
