Prerequisites · "what you need" header for academy tutorials

Wat je nodig hebt

  • Een werkend Woo-register. Volg eerst Een Woo-register opzetten als je dat nog niet hebt.
  • Een publicatie-object met bekende UUID.
  • curl, basisauth, en een paar testbestanden.
React / MDX usage

Mount in MDX as <Prerequisites/>

Top-of-tutorial card listing what the reader needs before starting. Composed from <HexCard icon="clipboard">; each item uses an orange hex bullet.

Import & example

import {Prerequisites, PrerequisiteItem} from '@conduction/docusaurus-preset/components';

<Prerequisites title="Wat je nodig hebt">
  <PrerequisiteItem>
    Een werkend Woo-register. Volg eerst <a href="...">Een Woo-register opzetten</a>
    als je dat nog niet hebt.
  </PrerequisiteItem>
  <PrerequisiteItem>Een publicatie-object met bekende UUID.</PrerequisiteItem>
  <PrerequisiteItem><code>curl</code>, basisauth, en een paar testbestanden.</PrerequisiteItem>
</Prerequisites>
