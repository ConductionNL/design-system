The component is <AgentTrace lines=[{kind, text, bullet, muted, indent}] cursor mode />. Line kinds: expand (gray rail + "+N lines"), continuation (indented muted continuation of a tool call), note (primary line with a coloured bullet: done = white, info = workspace-blue, orange = KNVB, mint = mint-300), tool (lavender-tinted tool call), status (orange "ing…" with animated dots).
