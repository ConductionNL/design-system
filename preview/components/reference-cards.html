React / MDX usage

Mount in MDX as <ReferenceCard/>

Used on partner-detail pages and the /references section. Pair with <ReferenceGrid/> for the 3-up layout.

Import & example

import {ReferenceGrid, ReferenceCard} from '@conduction/docusaurus-preset/components';

<ReferenceGrid columns={3}>
  <ReferenceCard
    sector="Gemeente"
    name="Gemeente Rotterdam"
    what="WOO portal in production sinds Q2 2025..."
    stack={['OpenCatalogi', 'OpenRegister', 'OpenConnector']}
  />
</ReferenceGrid>

Props

  • sector the sector tag at the top
  • name customer display name
  • what body paragraph describing what was shipped
  • stack string array of apps used
