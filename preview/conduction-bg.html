Always pointy-top · never rotate
Do
Pointy-top, vertex up. The huisstijl rule applies to every hex, decoration included. Movement comes from parallax and opacity — never from rotation.
Don't
Tilted hexes feel improvisational. The hex is the brand mark — keep it stable, vertex up. If a scene feels too rigid, vary size and depth, not angle.
How to use
<!-- 1. Drop the layer as the first child of a position:relative + overflow:hidden parent --> <section class="cta-banner"> <div class="conduction-bg" aria-hidden="true"></div> <h2>Pick an app. Install it. Done.</h2> … </section> /* 2. Copy the CSS */ .conduction-bg { position: absolute; inset: 0; overflow: hidden; pointer-events: none; } .conduction-bg .hex { position: absolute; clip-path: var(--hex-pointy-top); will-change: transform, opacity; transform: translate(-50%, -50%); animation: hex-breathe var(--dur, 14s) ease-in-out var(--delay, 0s) infinite; } @keyframes hex-breathe { 0%, 100% { opacity: var(--opa-min, 0.04); } 50% { opacity: var(--opa-max, 0.18); } } @media (prefers-reduced-motion: reduce) { .conduction-bg .hex { animation: none; opacity: var(--opa-max, 0.12); } } /* 3. Hydrate with the script — see <script> tag at the bottom of this page. It builds 18 hexes per .conduction-bg, drives parallax on scroll, and gates work behind IntersectionObserver. */
Anatomy
Parallax
translateY = -offset · depth · 0.4
Offset is the section's centre vs. viewport centre. Depth scales the move. Tier 0.10 hexes drift 4× slower than tier 0.85 — the small ones feel distant.
Breath
Each hex's opacity ease-in-outs between --opa-min and --opa-max on an 11–18 s cycle. Random negative delays so they're already mid-cycle on load — the field never pulses in unison.
Performance
IntersectionObserver gates the rAF scroll handler — zero work when the section is offscreen. prefers-reduced-motion disables both parallax and breath; hexes still render statically.
Rules
Do
  • Use it on cobalt CTAs, hero plates, and section closers — places that already feel like brand surface
  • Keep every hex pointy-top — depth comes from size + opacity, never from rotation
  • Place exactly one orange accent per scene; the rest are white at 0.03–0.20 opacity
  • Cap the field at ~18 hexes — past that the breath stops feeling calm
  • Always set pointer-events: none on the layer so foreground actions stay clickable
  • Honour prefers-reduced-motion — the layer is decorative, never load-bearing
Don't
  • Don't rotate hexes for "movement" — that breaks the huisstijl
  • Don't push opacity above ~0.25; the field is backdrop, not pattern
  • Don't stack two conduction-bg layers on the same section — one field per surface
  • Don't combine with the cut-corners motif on the same surface — pick one
  • Don't use it on light surfaces; the white-on-cobalt contrast is the whole effect
  • Don't speed up the breath cycle below ~10 s — the field starts to flicker
