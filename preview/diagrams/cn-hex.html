Why we built it this way

The atom carries the brand

Every hex on every Conduction surface goes through this one element. If the rules live here, they hold everywhere automatically.

  • Colours by name, never hex. The API rejects off-palette values implicitly. You cannot accidentally ship a teal hex; the attribute does not accept it.
  • Pointy-top is hard-coded. The clip-path polygon is not a prop. There is no flat-top variant. The brand says one orientation; the component says one orientation.
  • Slots over props for content. Label and icon are slotted in, so any HTML, any SVG, any image works. No string-only API to grow.
  • Interactive is a single attribute. Set it once and you get the cursor, the hover scale, the focus ring, the keyboard activation. No need to remember six things; the component owns the affordance.
  • No build step. Drop a script tag, write a hex. The design-system can use this today; the website can use it tomorrow without changing a line.
  • Shadow DOM isolates styles. Page CSS cannot reach inside and break the hex. The brand stays brand.
Uses No dependencies. This is the foundational primitive.
