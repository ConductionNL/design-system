Outcomes · "what you'll learn" header for academy tutorials

Wat je leert

  • Vier upload-modes voor documenten bij een Woo-publicatie, van klein naar groot.
  • Wanneer je JSON+base64, multipart of WebDAV chunked upload gebruikt.
  • Hoe je bestanden publiceert, depubliceert en weer opruimt.
React / MDX usage

Mount in MDX as <Outcomes/>

Top-of-tutorial card listing what the reader will have built or learned by the end. Goes right after the lede paragraph and before <Prerequisites/>. Composed from <HexCard icon="lightbulb">; each item uses an orange checkmark.

Import & example

import {Outcomes, Outcome} from '@conduction/docusaurus-preset/components';

<Outcomes title="Wat je leert">
  <Outcome>Vier upload-modes voor documenten, van klein naar groot.</Outcome>
  <Outcome>Wanneer je JSON+base64, multipart of WebDAV chunked upload gebruikt.</Outcome>
  <Outcome>Hoe je bestanden publiceert, depubliceert en weer opruimt.</Outcome>
</Outcomes>

Props

  • title string (default "What you'll learn")
  • children <Outcome> nodes
  • className string
