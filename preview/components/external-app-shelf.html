External app shelf · third-party open-source apps integrated as ExApps
Open source we use, not own

The Nextcloud workspace, the apps we don't write.

Each ExApp is packaged for the Nextcloud app store, so installing one feels the same as installing any other Nextcloud app. Sovereignty without choice is just a slogan; pick the engine, keep the workspace.

React / MDX usage

Mount in MDX as <ExternalAppShelf/>

Card grid for external open-source apps that Conduction integrates into the workspace as ExApps. Each card carries the project's brand-coloured logo tile, name, meta tag, and a one-sentence description. Used on /connext to show "we don't own these projects, we provide the interface."

Import & example

import {ExternalAppShelf} from '@conduction/docusaurus-preset/components';

<ExternalAppShelf
  apps={[
    {
      name: 'OpenTalk',
      meta: 'ExApp · Video',
      brandColor: '#00485C',
      desc: 'Open-source video conferencing.',
      href: 'https://opentalk.eu/',
      icon: <svg viewBox="0 0 24 24">...</svg>,
    },
    /* ... */
  ]}
/>

Props

  • apps array of {name, meta, brandColor, desc, href, icon}
  • brandColor per-app hex tile fill (the project's own brand colour)
  • icon SVG element rendered white inside the brand-coloured hex
  • href outbound link, opens in new tab

Cards lift on hover; arrow turns KNVB orange. Grid is responsive (auto-fill, min 280px).
