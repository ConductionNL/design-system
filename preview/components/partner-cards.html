React / MDX usage

Mount in MDX as <PartnerCard/>

Three tiers (partner, certified, strategic). Pair with <PartnerGrid/> and drop a <BecomePartner/> tile in the trailing slot for the directory CTA.

Import & example

import {PartnerGrid, PartnerCard, BecomePartner} from '@conduction/docusaurus-preset/components';

<PartnerGrid columns={3}>
  <PartnerCard
    href="/partners/acato"
    tier="strategic"
    name="Acato"
    logo="/img/partners/acato.png"
    summary="Nederlandse open-source specialist..."
    apps={['OpenRegister', 'OpenCatalogi', 'DocuDesk']}
  />
  <BecomePartner href="/partners/become" />
</PartnerGrid>

Props

  • tier partner | certified | strategic
  • name partner display name
  • logo img src for the logo plate
  • summary body text (string or ReactNode)
  • apps string array of app names
  • href partner-detail link
